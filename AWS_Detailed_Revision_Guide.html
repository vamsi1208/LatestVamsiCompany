Step 1 - AWS Account Creation

You created an AWS account and received AWS credits.

Key concepts:

  • Root User = Account owner.
  • Root user should not be used for daily development.
  • Use IAM users for development activities.
Step 2 - Understanding Regions

AWS resources exist in Regions.

  • us-east-1 = Virginia
  • us-west-2 = Oregon
  • ap-southeast-2 = Sydney
  • ap-south-2 = Hyderabad

For Bedrock learning we selected us-east-1.

Step 3 - IAM User Creation

Step 8 - S3 Fundamentals

S3 = Simple Storage Service.

Hierarchy:

AWS Account
  ↓
S3 Bucket
  ↓
Folders (optional)
  ↓
Objects (files)

Your bucket:

demo-bucket-397760697907-us-west-2-an
Step 9 - First S3 Upload

Create file:

echo Hello AWS > test.txt

Upload:

aws s3 cp test.txt s3://demo-bucket-397760697907-us-west-2-an/

Result:

  • test.txt successfully uploaded
  • Visible in AWS Console
Useful S3 Commands 
aws s3 ls

aws s3 ls s3://bucket-name/

aws s3 cp test.txt s3://bucket-name/

aws s3 cp s3://bucket-name/test.txt .

aws s3 rm s3://bucket-name/test.txt

aws s3 sync reports/ s3://bucket-name/reports/
